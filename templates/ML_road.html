(a) Supervised learning

TYPES



(i) Regression Algorithms

1.Linear Regression: A linear regression model tries to fit a line that best describes the relationship between the input variables and the output variable.

2.Polynomial Regression: A polynomial regression model uses higher degree polynomial functions to capture more complex relationships between input and output variables.

3.Ridge Regression: A ridge regression model is a linear regression model with an additional L2 regularization term in the cost function to prevent overfitting.

4.Lasso Regression: A lasso regression model is a linear regression model with an additional L1 regularization term in the cost function to prevent overfitting.

5.Support Vector Regression: A support vector regression model is based on the support vector machine algorithm and is used to predict continuous variables.

(ii) Classification Algorithms

1.Logistic Regression: A logistic regression model is used for binary classification problems where the output variable is either 0 or 1.

2.Naive Bayes: A Naive Bayes classifier is based on the Bayes theorem and is commonly used in text classification.

3.Decision Trees: A decision tree classifier is a tree-like model that partitions the input space into subsets based on the values of input variables and predicts the output variable based on the majority class of the corresponding subset.

4.Random Forests: A random forest classifier is an ensemble of decision trees, where each tree is trained on a random subset of the input variables and the final output is determined by majority voting.

5.Support Vector Machines: A support vector machine classifier is used to classify data by finding the best hyperplane that separates the classes.

TYPES

1.Q-learning: Q-learning is a model-free, off-policy reinforcement learning algorithm that learns to estimate the optimal action-value function for a given environment. It is based on the Bellman equation and uses an epsilon-greedy policy to explore the environment.

2.Deep Q-Networks (DQN): DQN is an extension of Q-learning that uses a neural network to approximate the action-value function. It also uses experience replay and target networks to stabilize the learning process.

3.Policy Gradient: Policy Gradient is a model-free, on-policy reinforcement learning algorithm that learns to optimize the policy directly. It uses gradient ascent to update the policy parameters and typically uses a neural network to parameterize the policy.

4.Actor-Critic: Actor-Critic is a model-based, on-policy reinforcement learning algorithm that combines the advantages of value-based and policy-based methods. It learns to estimate both the value function and the policy, and uses these estimates to update the parameters.

5.Monte Carlo Tree Search (MCTS): MCTS is a model-based, tree-based reinforcement learning algorithm that is particularly effective for sequential decision-making problems such as games. It uses a search tree to explore the state space and estimate the value of each action.

(a)Artificial Nural Network (ANNs)

Data Preparation: The first step in building an ANN is to prepare the data. This involves cleaning, preprocessing, and normalizing the data to make it suitable for the neural network. The data should also be split into training, validation, and testing sets.

Model Architecture: The next step is to design the architecture of the neural network. This involves deciding on the number of layers, the number of neurons in each layer, the activation functions to be used, and the type of loss function to be used.

Initialization: The weights and biases of the neural network are randomly initialized before training begins. There are different methods for initialization, including random initialization, Xavier initialization, and He initialization.

Forward Propagation: During forward propagation, the input data is passed through the neural network, and the output of each layer is calculated. The output of the last layer is the predicted output.

Calculation of Loss: The predicted output is compared to the actual output to calculate the loss using the chosen loss function. The goal is to minimize the loss during training.

Backward Propagation: During backward propagation, the gradients of the loss function with respect to the weights and biases are calculated using the chain rule of calculus. These gradients are then used to update the weights and biases using an optimization algorithm such as stochastic gradient descent.

Repeat: Steps 4-6 are repeated for multiple epochs until the neural network converges to a solution.

Evaluation: Once the neural network is trained, it is evaluated on the testing set to assess its performance.

(b)Convolutional Neural Networks (CNNs) 6th Jan 2021

Data Preparation: The first step in building a CNN is to prepare the data. This involves cleaning, preprocessing, and normalizing the data to make it suitable for the neural network. The data should also be split into training, validation, and testing sets.

Model Architecture: The next step is to design the architecture of the neural network. This involves deciding on the number of convolutional layers, pooling layers, fully connected layers, the number of filters in each layer, and the size of the filter. The activation functions, dropout, and batch normalization can also be used in the layers of the network.

Convolutional Layers: The input data is passed through the convolutional layers. These layers have a set of learnable filters that can be used to extract relevant features from the input data. The filters slide over the input data in a sliding window fashion, producing a feature map.

Pooling Layers: The output of the convolutional layers is passed through the pooling layers. These layers reduce the spatial dimensions of the feature map by downsampling the output from the convolutional layers.

Flattening: The output of the pooling layers is flattened into a one-dimensional vector, which is then passed through the fully connected layers.

Fully Connected Layers: The fully connected layers are used to make predictions based on the extracted features from the input data. These layers have a set of weights that are learned during the training process.

Dropout and Batch Normalization: Dropout and Batch Normalization can be used to improve the performance of the CNN. Dropout is used to prevent overfitting by randomly dropping out some of the neurons during training. Batch Normalization is used to normalize the input to each layer, which can help to speed up the training process and improve the performance of the network.

Optimization: During training, the weights of the neural network are updated using an optimization algorithm such as stochastic gradient descent.

Evaluation: Once the CNN is trained, it is evaluated on the testing set to assess its performance.

Environment: Define the environment that the agent will interact with, including the observation space, action space, and reward function.

Model Architecture: Design the architecture of the neural network. This involves deciding on the number of layers, the type of layers, the number of neurons in each layer, and the type of activation functions to be used. The output layer should correspond to the number of actions available in the environment.

Initialization: The weights and biases of the DRL model are randomly initialized before training begins. There are different methods for initialization, including random initialization, Xavier initialization, and He initialization.

Agent-Environment Interaction: The agent interacts with the environment by observing the current state and selecting an action based on its policy. The action is passed to the environment, which returns a reward and the next state.

Replay Buffer: Store the observed transitions in a replay buffer. This allows the agent to learn from past experiences and break the correlation between consecutive observations.

Update: Periodically update the parameters of the DRL model using the data in the replay buffer. This involves sampling a batch of transitions from the replay buffer, computing the loss function, and updating the weights of the neural network using an optimization algorithm such as stochastic gradient descent.
